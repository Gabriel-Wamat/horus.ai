# nexus-ops-command-center-e2e

Projeto construído em workspace isolado pelo Horus.

## User stories selecionadas
- NX-01 - Shell navegavel e dashboard operacional
- NX-02 - Triagem de incidentes com board, filtros e detalhe
- NX-03 - Mapa de ativos e dependencias com modo simplificado
- NX-04 - Automacoes operacionais com aprovacao e execucao auditavel
- NX-05 - Workbench de assistente com chat, evidencia e preview
- NX-06 - Relatorios, preferencias e configuracoes de provider

## Estrutura

- `src/index.html`: scaffold visual inicial.
- `docs/user-stories`: user stories e specs versionadas dentro do projeto.
- `.horus-project.yaml`: contrato de comandos, write roots e perfis dos agentes.
