# Horus Tools Workspace

This folder is reserved for project-owned tools and reusable tool documentation.

Current status:

- Runtime CLI execution is not implemented here yet.
- Production runtime code must remain inside the owning app/package boundary unless a spec explicitly promotes a reusable tool from this folder.

Mandatory rule:

- The first implementation priority for the upcoming tool work is proving real CLI capability with safe command execution, process lifecycle control, command evidence, and tests.
